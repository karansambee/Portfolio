Java game using CityEngine: Forest Treasures

This is my 2D Java platformer game created using the City Engine.

The aim of the game is to escape the traps and dangers of the forest and escape through the portals, on your way collecting the array of coins scattered over the levels.

The main challenge I faced during the development of the project was time management. Particularly, estimating how long it would take to implement a specific feature as unforeseen bugs and gaps in my knowledge slowed down my coding process.

The lessons I have learned from this coursework are:
- The importance of the readablity of your code when fixing bugs that span across multiple classes
- How much time is saved through inhertiance since you are duplicating less code. For example, my GameLevel allowed me to created 4 levels easily and my Coin superclass allowed me to quickly create bronze, silver and gold coins efficently.
